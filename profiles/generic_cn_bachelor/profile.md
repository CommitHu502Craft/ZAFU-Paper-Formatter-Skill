# Generic CN Bachelor Profile

This is a conservative fallback profile for thesis-like Chinese undergraduate documents that do not strongly match a school template.

## Intended use

Use this profile when:
- the document looks thesis-like
- the school template is unavailable
- only generic low-risk normalization should be attempted

## Default constraints

- prefer `audit-only` when structural risk is high
- allow `conservative-repair` only for safe-subset body, heading, caption, table, and page-geometry fixes
- avoid template-specific front-matter assumptions

## Current status

This profile is a scaffold for future expansion. It intentionally avoids school-specific promises.
